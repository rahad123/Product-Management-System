# Product Management System

## Requirements

- Node.js ([Download](https://nodejs.org/en))
- TypeScript
- Express.js
- NPM (Will be download with Node.js)

## Installation

- Copy .env.example to .env

```
 cp .env.example .env
```

### Run

```
npm run dev
```

## Tests

- For testcase you can run

```
npm run test
```
